## 安装 FFmpeg

### macOS

推荐使用 Homebrew：

```bash
brew install ffmpeg
```

### Windows

推荐两种方式：

- 使用 WSL，然后在 Linux 环境中安装 FFmpeg。
- 使用 Git Bash 执行本教程中的 sh/bash 脚本，并使用 Windows 版 FFmpeg。

### 检查安装

```bash
ffmpeg -version
ffprobe -version
ffplay -version
```

如果 `ffplay` 不可用，不影响大部分教程任务；它主要用于快速预览媒体文件。

## 路径和覆盖规则

- 输入文件放在 `examples/input/`。
- 输出文件放在 `examples/output/`。
- 不直接覆盖源文件。
- 需要覆盖输出文件时显式使用 `-y`。
- 需要避免覆盖时使用 `-n`。
