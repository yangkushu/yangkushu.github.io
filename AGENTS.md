# 项目协作说明

## 文章编辑上下文

编辑单篇文章时，应先检查并读取对应的文章上下文或规格文件：

- 单文件文章：如果同目录存在与文章同 basename 的 `*.context.md` 或 `*.spec.md`，先读取它。
- 目录型文章：如果文章目录下存在 `context.md`，先读取它。

文章上下文或规格文件只约束对应文章，不作为项目级规范。

## 公开仓库边界

本仓库只保留网站发布所需内容，包括 Hugo 配置、文章正文、静态资源、主题、构建流程和公开协作说明。

文章规格、执行计划、评审模板、运维检查记录、辅助脚本等非发布必需资料，应放在私有工作仓库中；确认适合公开后，再整理为文章正文或公开文档同步到本仓库。
